# 规则库变更日志

## [1.0.0] - 2025-12-16

### 新增
- `SEC-001`: 禁止使用 eval()
- `SEC-002`: 禁止使用 exec()
- `SEC-003`: 禁止使用 os.system()
- `SEC-004`: 禁止使用 subprocess.call(shell=True)
- `NET-001`: 网络请求必须设置超时
- `NET-002`: 网络请求应有重试机制
- `NET-003`: 禁止硬编码 URL
- `NET-004`: HTTPS 优先
- `FILE-001`: 文件操作必须使用 with 语句
- `FILE-002`: 禁止硬编码文件路径
- `FILE-003`: 文件删除需要确认
- `FILE-004`: 路径拼接使用 pathlib
- `DB-001`: 禁止 SQL 字符串拼接
- `DB-002`: 数据库连接必须关闭
- `DB-003`: 事务必须提交或回滚
- `DB-004`: 禁止 SELECT *

### 说明
这是规则库的初始版本，包含了最基础的安全和最佳实践规则。
后续版本将根据假绿事件分析结果持续改进。
